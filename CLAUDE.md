1. **Sempre** leia `.docs/Agents.md` antes de começar qualquer tarefa
2. **Sempre** consulte `.docs/Phases.md` para saber a fase atual
3. **Sempre** consulte `.docs/CurrentState.md` para entender o que funciona agora
4. **Sempre** consulte `.docs/DecisionLog.md` para ver decisões já tomadas
5. **Nunca** pule uma fase ou misture escopos de fases diferentes
6. **Sempre** atualize os docs ao tomar decisões ou mudar o estado do sistema
7. **Nunca** apague entradas do `DecisionLog.md` — ele é append-only
8. **Sempre** consulte `.docs/ProjectScope.md` para entender os requisitos de negócio
